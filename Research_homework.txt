1. What is .gitignore? You can write the answer either in Korean or English.
Ans) git으로 관리하지 않을 파일을 설정하여, git이 추적하지 않도록 한다. 예를 들어 파일에 크기가 지나치게 크거나, 개인적인 정보가 포함되어 있는 경우에는 git이 따로 해당 파일을 추적하지 않도록 할 수 있다.
2. Why do Github users need an SSH key pair? (Users can use either SSH key pair or github account info.) Please write down a brief explanation of SSH key.
Ans) SSH는 Secure Shell의 약자로, 개인정보 노출의 위험 때문에 도입되었다. SSH key를 사용할 경우, 원격 인터넷에 접속하는 과정에서 통신이 노출되더라도 암호화된 정보로 표현되어 해당 정보를 확인할 수 없도록 할 수 있다.
즉, 암호화와 개인정보 보호의 측면 때문에 사용한다.
